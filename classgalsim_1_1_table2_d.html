Detailed Description

template<class V, class A>
class galsim::Table2D< V, A >

A class to represent lookup tables for a function z = f(x, y).

A is the type of the argument of the function. V is the type of the value of the function.

Requirements for A,V: A must have ordering operators (< > ==) and the normal arithmetic ops (+ - * /) V must have + and *.
