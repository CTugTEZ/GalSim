Detailed Description

Class to describe optical aberrations in terms of Zernike polynomial coefficients.

Input aberration coefficients are assumed to be supplied in units of wavelength, and correspond to the Zernike polynomials in the Noll convention defined in Noll, J. Opt. Soc. Am. 66, 207-211(1976). For a brief summary of the polynomials, refer to http://en.wikipedia.org/wiki/Zernike_polynomials#Zernike_polynomials.

Parameters

spherSpherical aberration in units of reference wavelength. [default: 0]
aberrationsOptional keyword, to pass in a list, tuple, or NumPy array of aberrations in units of reference wavelength (ordered according to the Noll convention), rather than passing in individual values for each individual aberration. Note that aberrations[1] is piston (and not aberrations[0], which is unused.) This list can be arbitrarily long to handle Zernike polynomial aberrations of arbitrary order.
lam_0Reference wavelength in nanometers at which Zernike aberrations are being specified. [default: 500]

Compute wavefront due to optical phase screen.

Wavefront here indicates the distance by which the physical wavefront lags or leads the ideal converging Gaussian reference spherical wave.

Parameters
