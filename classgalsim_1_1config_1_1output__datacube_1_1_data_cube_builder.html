Build the images.

A point of attention for DataCubes is that they must all be the same size. This function builds the first image alone, finds out its size and then forces all subsequent images to be the same size.

Parameters
configThe configuration dict for the output field.
baseThe base configuration dict.
file_numThe current file_num.
image_numThe current image_num.
obj_numThe current obj_num.
ignoreA list of parameters that are allowed to be in config that we can ignore here. i.e. it won't be an error if they are present.
loggerIf given, a logger object to log progress.
Returns
a list of the images built
