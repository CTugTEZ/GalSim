Detailed Description

The return type of Angle.rad.

A special type that works in most ways as a float, but which allows the use of Angle.rad() as a function rather than a property, but raising a deprecation warning.

If you have trouble using this type as a float, you can write 

>>> x = float(angle.rad)

to explicitly turn it into a regular float. This won't be necessary in version 2.0 (and it's probably not ever necessary now).
