Detailed Description

Dictionary class that allows for easy initialization and refs to key values via attributes.

NOTE: Modified a little from Jim's bot.git AttributeDict class so that tab completion now works in ipython since attributes are actually added to dict.

HOWEVER this means the dict attribute has been redefined to be a collections.defaultdict() so that Jim's previous default attribute behaviour is also replicated.
