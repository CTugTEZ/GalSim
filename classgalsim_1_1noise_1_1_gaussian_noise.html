Detailed Description

Class implementing simple Gaussian noise.

This is a simple noise model where each pixel in the image gets Gaussian noise with a given sigma.

Initialization

Parameters
rngA BaseDeviate instance to use for generating the random numbers.
sigmaThe rms noise on each pixel. [default: 1.] 
Methods
-------

To add noise to every element of an image, use the syntax `image.addNoise(gaussian_noise)`.

    noise.rng           # The internal random number generator (read-only)
    noise.sigma         # The value of the constructor parameter sigma (read-only)

Add noise to an input Image.

Calling 

>>> noise.applyTo(image)

On output the Image instance image will have been given additional noise according to the current noise model.

Note: This is equivalent to the alternate syntax 

>>> image.addNoise(noise)

which may be more convenient or clearer.
