Detailed Description

The TanWCS type needs special handling to get the kwargs, since the TanWCS function takes an AffineTransform as one of the arguments, so we need to build that from dudx, dudy, etc.

We also need to construct a CelestialCoord object for the world_origin, which we make from ra, dec paramters.
