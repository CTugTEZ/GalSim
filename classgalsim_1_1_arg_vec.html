Detailed Description

template<class A>
class galsim::ArgVec< A >

A class to represent an argument vector for a Table or Table2D.

Basically a std::vector with a few extra bells and whistles to deal with potentially equally-spaced arguments, upper and lower slop, and fast indexing.
